R.1  Account Management

	R.1.1 Login

		Description : User can login by providing email and password.
		Input : Email,password
		Output : If email and password correct then user redirect to home           
			     page else error message print username or password is 
			     incorrect.
	R.1.2 Create Account

		Description : User can create account.
		Input : Name, email, contact_no and password.
		Output : Success message.
